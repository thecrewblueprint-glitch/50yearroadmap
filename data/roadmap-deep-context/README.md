# Roadmap Deep Context

This folder is reserved for roadmap-specific deep-context material.

These files are not generic source digests. They carry the deeper mission, emotional meaning, identity, long-range intent, values, and human reasons behind the 50-year roadmap.

## Purpose

Use this category for conversations and exports that primarily discuss:

- the 50-year roadmap as a life/project architecture
- the meaning behind the roadmap, not only tasks
- Homes for Hands and the Homestead Network
- Deadhang, The Crew Blueprint, Production Atlas, and other projects as parts of a larger life system
- emotional drivers, values, fears, obligations, hopes, and personal stakes
- long-term milestones, legacy planning, and identity-level direction
- why a project matters, not just what needs to be built

## Default Treatment

Roadmap deep-context files should be handled differently from normal operational data:

1. **Preserve meaning first.** Do not flatten emotional or mission context into generic task bullets too early.
2. **Summarize carefully.** Extract themes, values, tensions, long-range decisions, and operating principles.
3. **Separate public-safe output from private meaning.** Public roadmap files should not expose intimate, sensitive, or unnecessary personal detail.
4. **Do not overwrite operational trackers directly.** Convert deep-context material into structured digest notes first.
5. **Keep source lineage.** Every extracted claim should point back to the originating file/conversation when possible.

## Suggested File Naming

Use this pattern:

```text
YYYY-MM-DD-source-topic-deep-context.md
```

Examples:

```text
2026-07-06-gemini-homes-for-hands-deep-context.md
2026-07-07-gemini-roadmap-values-deep-context.md
2026-07-07-gemini-homestead-governance-deep-context.md
```

## Extraction Format

When extracting these files into digest form, use:

```markdown
# Deep Context Digest — YYYY-MM-DD — Topic

## Source Type
Gemini roadmap deep-context conversation.

## Core Meaning
What this file says about the roadmap's deeper purpose.

## Emotional / Mission Context
What matters emotionally, ethically, or personally.

## Roadmap-Relevant Decisions
Concrete decisions that should influence roadmap structure.

## Values and Operating Principles
Rules, values, or standards that should shape execution.

## Projects Affected
- pillar-1 — Deadhang Labor LLC
- pillar-2 — The Crew Blueprint
- pillar-3 — Production Atlas
- pillar-6 — Homestead Network / Homes for Hands
- pillar-7 — Personal Operations / Roadmap System

## Blockers / Tensions
Unresolved questions, fears, contradictions, or governance issues.

## Public-Safe Summary
A version that can be used in the repo without exposing sensitive personal material.

## Private / Do Not Surface Publicly
Notes that should remain private or be abstracted before public use.
```

## Homes for Hands Classification

Homes for Hands belongs under:

```text
pillar-6 — Homestead Network
```

It should be treated as a nonprofit homestead/respite project for stagehands and travelers who need rest between gigs, with a possible permanent landing space for those who need one.

## Relationship to `data/raw-reports`

`data/raw-reports` remains the location for watcher-ready operational digests.

This folder is for deeper roadmap context before it is converted into watcher-ready digest entries. When a deep-context file produces actionable roadmap items, create or update a digest in `data/raw-reports/` with public-safe wording and clear pillar mapping.
