# 08 — Brand

**Status:** Documented (exact values + assets referenced to their sources).

## Identity

- **Colors:** navy & gold. The **exact hex values are defined in the
  deadhanglaborllc.com site CSS**, which is the source of truth — not hardcoded
  here, so the brand stays in sync with the live site.
- **Name / mark:** Deadhang Labor LLC; a company logo used in the front
  "name-tag" position, with a steel-tower/crane illustration concept for backs
  (used in merch — see doc 09).
- **Look:** clean, professional, industrial — matches the live-event/rigging trade.

## Voice

- **Grounded and honest.** No overstatement: portfolio and materials describe
  structures **assisted** rather than claiming sole responsibility, and avoid
  implying employment relationships that didn't exist.
- **Trade-authentic, with humor** where appropriate — the merch line leans into
  stagehand humor (e.g. "Gravity Is Always Hiring", "Professional Box Pusher",
  "Living Out of a Pelican Case").

## Standards

- Portfolio captions must be accurate and non-inflated (venue credibility over
  hype).
- Consistent navy/gold across website, invoices (branded PDF), and merch.

## Design system (per-project CSS schemas)

Each project maintains **its own CSS/brand schema** — they are intentionally
separate, so don't share one stylesheet across them:

- **Deadhang Labor LLC** — the deadhanglaborllc.com site CSS (source of truth for
  Deadhang's navy/gold values).
- **The Crew Blueprint** — its own schema.
- **This roadmap / command center** — its own theme (`styles.css`).
- **Production Atlas** — has current light + dark CSS; a new schema is planned.

## Asset locations (referenced)

- Logo, portfolio photography, and design assets are maintained in the
  `Contentrepo` / portfolio material, not duplicated here.
