About this project

Production Atlas is a project of Deadhang Labor LLC, a live-event production labor company. It was built as a public-safe static research resource to help production workers find festival opportunities, compare timing and geography, review public employer routes, and plan what to research next.

What Production Atlas is

Production Atlas is a public-facing work map for live-event production. It organizes festival dates, locations, approximate production windows, public producer or promoter information, employer routes, IATSE/local research guidance, source references, and map/calendar planning views so contractors can decide which events are worth researching.

The current app is static. It has no public visitor login, backend database, payment flow, private contact storage, or scraping automation.

Built for public research

Production Atlas uses public-facing event, company, source, and directory information. It does not publish non-public work details, personal contact information, pay rates, lodging details, private referrals, or rumors.

Built for planning

The opportunities, calendar, map, employers, IATSE, sources, and direct-URL schedule pages help workers compare timing, region, and public employer routes before deciding where to focus outreach.

Built for source review

Source links are centralized on the Sources page so opportunity and employer screens stay readable while public references remain available for review.

Supplemental pages

Branches, Matrix, and Analytics are retained as supplemental review pages. Analytics includes an action-first research queue for audit/scouting work, but that queue is not a primary public filter or job-offer system.

What it is not

Production Atlas is not a job board, hiring guarantee, private contact database, labor broker, legal advisor, travel agent, dispatch system, or substitute for direct verification with official event, employer, venue, or local-jurisdiction sources.
